# Basics on Python

---
## Write Python Codes

* Comments: ```#```
* Line Continuation: ```\```
* Indent: ```[Tab]``` or ```spaces(2 or 4)```
* Simple Quotation: ```' '``` or ```" "```
* Line-over Quotation(for ```docstring```): ```""" """``` or ```''' '''```
* Assignment: ```=```
